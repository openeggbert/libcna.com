Tutorial 88: Nova-3D: Building a Game Engine on CNA

CNA — C++ XNA 4.0 reimplementation

What is Nova-3D?

Nova-3D is an open-source 3D game engine that uses CNA as its rendering and platform runtime. Where CNA provides the low-level XNA-compatible API, Nova-3D adds higher-level abstractions: a scene graph, entity-component system (ECS), physics integration, and scripting support. Nova-3D is part of the OpenEggbert project.

CNA as runtime foundation

Nova-3D initializes a Game subclass and calls into CNA for all rendering, input, and audio. From CNA's perspective, Nova-3D is just a game that uses GraphicsDevice, SpriteBatch, Model, and SoundEffect. Nova-3D never bypasses CNA's API — it builds on top of it.

What CNA provides

  • Window creation and event loop (Game, GameComponent)
  • 2D and 3D rendering (SpriteBatch, BasicEffect, Model, VertexBuffer, IndexBuffer)
  • Input devices (Keyboard, Mouse, GamePad, TouchPanel)
  • Audio playback (SoundEffect, SoundEffectInstance)
  • Math library (Vector2, Vector3, Matrix, Quaternion, BoundingBox, BoundingSphere)
  • Cross-platform file I/O (ContentManager, storage path helpers)
  • Render targets and post-processing (RenderTarget2D)

What Nova-3D adds

  • Scene graph with parent-child transform hierarchy
  • Entity-component system (ECS): Entity, Component, System
  • Physics: Box2D (2D) and Bullet Physics (3D) integration
  • Scripting: Lua or AngelScript bindings
  • Asset pipeline: batch import tool for OBJ/FBX/PNG/OGG
  • Editor: an in-progress scene editor built with ImGui

Simple entity component system built on CNA Game class

#include "Microsoft/Xna/Framework/Game.hpp"
#include "Microsoft/Xna/Framework/Graphics/GraphicsDeviceManager.hpp"
#include "Microsoft/Xna/Framework/Graphics/SpriteBatch.hpp"
#include "Microsoft/Xna/Framework/Vector2.hpp"
#include <vector>
#include <memory>
#include <string>

using namespace Microsoft::Xna::Framework;
using namespace Microsoft::Xna::Framework::Graphics;

// --- Minimal ECS ---
struct Component { virtual ~Component() = default; };

struct TransformComponent : Component {
    Vector2 position;
    float   rotation = 0.0f;
    float   scale    = 1.0f;
};

struct SpriteComponent : Component {
    std::string texturePath;
    Color       tint = Color::White;
};

struct Entity {
    template<typename T, typename... Args>
    T& AddComponent(Args&&... args) {
        auto comp = std::make_unique<T>(std::forward<Args>(args)...);
        T& ref = *comp;
        components_.push_back(std::move(comp));
        return ref;
    }
    template<typename T>
    T* GetComponent() {
        for (auto& c : components_) {
            if (auto* p = dynamic_cast<T*>(c.get())) return p;
        }
        return nullptr;
    }
    std::vector<std::unique_ptr<Component>> components_;
};

// --- Nova-style game using CNA ---
class Nova3DGame final : public Game {
public:
    Nova3DGame() : graphics_(this) {}

protected:
    void LoadContent() override {
        spriteBatch_ = std::make_unique<SpriteBatch>(getGraphicsDeviceProperty());

        // Create a player entity
        auto& player = entities_.emplace_back();
        auto& tf = player.AddComponent<TransformComponent>();
        tf.position = Vector2(400, 300);
        auto& sp = player.AddComponent<SpriteComponent>();
        sp.texturePath = "assets/player.png";

        // Load textures for all sprite components
        for (auto& e : entities_) {
            auto* sc = e.GetComponent<SpriteComponent>();
            if (sc) {
                textures_.push_back(std::make_unique<Texture2D>(
                    sc->texturePath, getGraphicsDeviceProperty()));
            }
        }
    }

    void Update(const GameTime& gt) override {
        float dt = static_cast<float>(gt.ElapsedGameTime.TotalSeconds());
        // Update systems here (movement, physics, AI...)
        for (auto& e : entities_) {
            auto* tf = e.GetComponent<TransformComponent>();
            if (tf) tf->rotation += dt * 1.0f;  // rotate all entities
        }
    }

    void Draw(const GameTime&) override {
        auto& gd = getGraphicsDeviceProperty();
        gd.Clear(Color::CornflowerBlue);
        spriteBatch_->Begin();

        int texIdx = 0;
        for (auto& e : entities_) {
            auto* tf = e.GetComponent<TransformComponent>();
            auto* sc = e.GetComponent<SpriteComponent>();
            if (tf && sc && texIdx < (int)textures_.size()) {
                spriteBatch_->Draw(*textures_[texIdx++],
                                   tf->position, nullptr,
                                   sc->tint,
                                   tf->rotation,
                                   Vector2::Zero,
                                   tf->scale,
                                   SpriteEffects::None, 0.0f);
            }
        }
        spriteBatch_->End();
        gd.Present();
    }

private:
    GraphicsDeviceManager               graphics_;
    std::unique_ptr<SpriteBatch>        spriteBatch_;
    std::vector<Entity>                 entities_;
    std::vector<std::unique_ptr<Texture2D>> textures_;
};

int main() { Nova3DGame game; game.Run(); }

Contributing to Nova-3D

Nova-3D is hosted under the OpenEggbert organization on GitHub. Contributions are welcome in the scene graph, ECS, physics integration, and editor areas. The project requires a working CNA installation as a prerequisite. See the Nova-3D repository README for build instructions.
